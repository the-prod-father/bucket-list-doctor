## Usage Tips

1. **Test each prompt** - Generate a few variations to find the best style match
2. **Size considerations** - Icons should work well at both small (64px) and larger (128px) sizes
3. **Background** - Consider generating with transparent backgrounds for flexibility
4. **Consistency** - Try to maintain a consistent style across all 10 icons
5. **Color variations** - You may want to generate multiple color variations to match different card backgrounds
